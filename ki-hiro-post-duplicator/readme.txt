投稿・固定ページなどの記事一覧に「複製」を追加します。

== Description ==

「複製」をクリックすると、本文、抜粋、アイキャッチ画像、カテゴリー、タグ、カスタムフィールドなどを引き継いだ下書きを作成し、その編集画面を開きます。

== Installation ==

1. `ki-hiro-post-duplicator` フォルダーを `/wp-content/plugins/` に配置します。
2. WordPress 管理画面の「プラグイン」で有効化します。
3. 投稿一覧または固定ページ一覧の「複製」をクリックします。
